


Good Starting point for learning .net core using visual studio code editor

1) install visual studio code
    http://code.visualstudio.com/

2) install .net core tooling (select "command line /other" if don't already have it installed e.g via full visual studio )
    https://www.microsoft.com/net/core#windowscmd

3) install Visual studio code extension "C# for Visual Studio Code (powered by OmniSharp)" by microsoft
    click extensions button on left hand side of visual studio code ide (square one below debug) to bring up extension panel to search for it in marketplace
    Search for "c#" and it should on the list, then install it. 
    Some other interesting extensions are also listed but suggest you explore these a little later such as the CSharp2TS or scriptcsRunner

3) follow "hello, console app"  example at 
https://docs.microsoft.com/en-us/dotnet/articles/core/tutorials/using-with-xplat-cli
